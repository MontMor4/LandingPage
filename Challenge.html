Desafio da Landing Page

Uma landing page é uma página da web projetada especificamente para direcionar visitantes a realizar uma ação específica, como fazer uma compra, preencher um formulário ou inscrever-se em uma lista de e-mails. Ela é criada com o objetivo de capturar a atenção do visitante, apresentar uma oferta clara e persuasiva, e incentivar a conversão. Uma landing page é projetada de forma simplificada, com foco no conteúdo relevante e em direcionar o visitante para a ação desejada, visando obter altas taxas de conversão.

Seu desafio é criar uma landing page usando HTML e CSS. Sua landing page deve conter os seguintes elementos HTML e propriedades CSS:

Elementos HTML

  • <html>: Define o início e o fim do documento HTML.
  • <head>: Contém informações sobre o documento HTML, como o título da página, metadados e scripts.
  • <title>: Define o título da página que será exibido na barra de título do navegador.
  • <body>: Contém todo o conteúdo visível da página, como texto, imagens, links e elementos de formulário.
    • <header>: Define o cabeçalho de uma seção ou de todo o documento.
    • <section>: Define uma seção de conteúdo dentro de um documento HTML.
    • <nav>: Define uma seção de navegação.
    • <footer>: Define o rodapé de uma seção ou de todo o documento.
    • <h1> a <h6>: Define os cabeçalhos de diferentes níveis, onde <h1> é o mais importante e <h6> é o menos importante.
    • <p>: Define um parágrafo de texto.
    • <a>: Cria um link para outra página ou recurso. O atributo href especifica o destino do link.
    • <img>: Exibe uma imagem na página. O atributo src especifica o caminho ou URL da imagem.
    • <ul> e <ol>: Criam listas não ordenadas e ordenadas, respectivamente. Os itens da lista são definidos com a tag <li>.
    • <div>: Define uma divisão ou seção genérica em um documento HTML. É frequentemente usada para agrupar elementos e aplicar estilos CSS.
    • <span>: Define uma parte genérica de texto em um documento HTML. É frequentemente usada para aplicar estilos CSS ou manipular o texto com JavaScript.
    • <table>: Cria uma tabela com linhas <tr>, células de dados <td>, e células de cabeçalho <th>.
    • <form>: Cria um formulário interativo com campos de entrada, botões e outras ferramentas para coletar dados do usuário.
    • <input>: Define um campo de entrada de dados em um formulário. Pode ser usado para textos, senhas, seleção de opções, entre outros.
    • <button>: Cria um botão interativo que pode ser usado para enviar formulários ou acionar ações específicas no lado do cliente.

Propriedades CSS

  • color: Define a cor do texto.
  • font-family: Especifica a família de fontes a ser usada para o texto.
  • font-size: Define o tamanho da fonte do texto.
  • font-weight: Controla a espessura da fonte, como negrito (bold) ou normal.
  • text-align: Alinha o texto à esquerda, à direita, ao centro ou justificado.
  • background-color: Define a cor de fundo de um elemento.
  • margin: Define as margens externas de um elemento.
  • padding: Define o espaço interno entre o conteúdo de um elemento e suas bordas.
  • border: Define a borda de um elemento, especificando sua largura, estilo e cor.
  • width: Define a largura de um elemento.
  • height: Define a altura de um elemento.
  • display: Controla como um elemento é exibido.
  • position: Controla o posicionamento de um elemento.
  • float: Define o posicionamento horizontal de um elemento em relação a outros elementos.
  • opacity: Define a opacidade de um elemento.
  • transition: Permite criar efeitos de transição suaves em propriedades.
  • box-shadow: Adiciona sombra a um elemento.
  • text-decoration: Controla a decoração do texto.
  • justify-content: Alinha os itens de um container flexível ao longo do eixo principal.
  • align-items: Alinha os itens de um container flexível ao longo do eixo transversal.

Instruções adicionais

  • Não se preocupe tanto com construir a melhor Landing Page do mundo. Foque em utilizar os elementos citados e a sua criatividade.

  • Fique à vontade para explorar outras propriedades CSS e elementos HTML, além dos mencionados acima, se desejar.

  • Utilize o conteúdo e as imagens relevantes ao tema escolhido para tornar sua landing page mais atrativa e persuasiva.

  • Certifique-se de testar sua landing page em diferentes dispositivos e tamanhos de tela para garantir uma boa experiência de usuário.

  • Ao terminar, compartilhe sua landing page com a gente para obter feedback e melhorias.
